# FridgeBook

This is an application that keeps track of your kitchen and inventory. 
This application is used so that you can save money on ingredients and make sure items do not expire!

## Technologies used

* MySQL
* Sequelize
* Passport
* FullCalendar
* Spoonacular API
* Bootstrap
* CSS Animations
* ReactJs
* SweetAlert
